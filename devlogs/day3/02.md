# Authoring Workflow Draft

We spent part of the day on the content pipeline because the team needs to add lessons faster than engineers can ship UI updates. The first authoring format is simple, but it is enough to unblock curriculum work.

Current authoring goals:
- Keep lesson structure readable in plain text.
- Allow quiz data to live next to lesson content.
- Make it easy to attach interactive graph prompts to a section.

This should help us scale content without turning every lesson change into a code task.
