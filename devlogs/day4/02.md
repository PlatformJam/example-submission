# Content Import Pipeline

We improved the path from authored lesson content into the app. That should reduce friction for the final stretch, where content iteration will likely outpace component changes.

Pipeline goals now:
- Validate lesson structure before it reaches the UI.
- Catch missing quiz explanations and malformed interactive prompts early.
- Keep content updates low-risk so we can revise quickly after feedback.

This is the kind of infrastructure work that does not look flashy but saves time immediately.
