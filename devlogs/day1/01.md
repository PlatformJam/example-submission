# Product Shape Emerging

We spent the afternoon sketching the actual learning loop instead of talking in abstractions. The current loop is: pick a lesson, manipulate a graph, answer a few questions, then see progress reflected immediately.

Decisions made:
- Lessons will be short and interactive instead of long-form text pages.
- Quizzes will reinforce one concept at a time instead of bundling many ideas together.
- Visual graph state needs to update fast enough that experimentation feels natural.

This gives us a stable target for the first prototype.
