# Quiz Engine Prototype

The quiz system moved from paper design into a working prototype. It can now present a question, score an answer, and surface feedback immediately, which is enough for internal testing.

Completed today:
- Added support for multiple-choice question flows.
- Defined a consistent feedback pattern for right and wrong answers.
- Started tracking which concepts a learner struggles with most.

The next improvement is better hinting so mistakes feel useful instead of punishing.
