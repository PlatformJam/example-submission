# Onboarding Walkthrough

We added a clearer first-run experience so new users understand what LearnGraph is asking them to do. Early feedback suggested people were interested, but not always immediately oriented.

The onboarding pass now:
- Introduces the lesson, graph canvas, and quiz relationship.
- Sets expectations for short, interactive practice sessions.
- Gives users a clearer first action instead of dropping them into an empty view.

This should improve comprehension before the learner encounters any real difficulty.
