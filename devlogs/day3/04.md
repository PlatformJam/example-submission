# Internal Playtest Notes

We ran a small internal playtest and got useful feedback on pacing. People liked the interactive graph work, but they consistently asked for clearer framing before each exercise.

Main takeaways:
- Learners want to know the purpose of an interaction before touching the graph.
- Quiz explanations matter almost as much as quiz scores.
- The progress screen is more motivating when it highlights recent wins.

Tomorrow will focus on turning those notes into concrete product changes.
